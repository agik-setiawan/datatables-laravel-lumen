Datatable for Lumen, Laravel Query Builder

Requirement:

  • PHP >=7.0
  • Laravel/Lumen

How to use on Lumen:

  • go to root your application
  • run command "composer require agik/datatables-laravel-lumen"
  • go to bootstrap directory
  • add $app->register(Datatable\DataTableServiceProvider::class); in app.php
  • add class_alias(Datatable\Facades\DataTable::class, 'DataTable'); in app.php
